# 泰国投资报告审核系统

基于思维导图的参考资料完整性审核工具，用于《对外投资合作国别（地区）指南-泰国（2025年版）》的参考资料审核。

## 功能特点

### 🗺️ 思维导图视图
- 以树形结构展示报告章节
- 三级展开：母节点 → 链接URL → 链接描述
- 支持缩放、平移、搜索

### ✅ 审核功能
- **状态标记**: 每个section可标记为"已审核/待审核/需补充"
- **详细注释**: 添加文字注释说明需要补充的内容
- **内容对比**: 对比原文与参考资料内容
- **覆盖评估**: 标记信息覆盖程度（完整/部分/缺失）

### 📊 统计与导出
- 实时统计审核进度
- 导出完整审核报告
- 本地存储审核进度

## 技术栈

- **前端**: HTML5, CSS3, JavaScript
- **可视化**: D3.js
- **图标**: Font Awesome
- **部署**: GitHub Pages

## 使用说明

### 基本操作
- **点击节点**: 展开/折叠子节点
- **双击节点**: 在侧边栏显示详情
- **鼠标滚轮**: 缩放导图
- **拖拽空白区域**: 平移导图

### 审核流程
1. 点击section节点查看详情
2. 阅读原文内容和参考链接
3. 点击"对比"按钮比较内容
4. 选择审核状态（已审核/待审核/需补充）
5. 添加注释说明需要补充的内容
6. 点击"保存进度"按钮

### 快捷键
- `Esc`: 关闭侧边栏和模态框
